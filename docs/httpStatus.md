# HTTP Status code
HTTP Status Code คือรหัสที่ server ส่งกลับไปเพื่อบอกผลลัพธ์ของ request ว่าสำเร็จ ล้มเหลว หรือผิดพลาดแบบไหน

### สรุปง่าย ๆ  
| Code | Name                 | ใช้เมื่อ                              |
| ---- | -------------------- | ------------------------------------- |
| 200  | OK                   | request สำเร็จ เช่น อ่านข้อมูลได้     |
| 201  | Created              | สร้าง resource สำเร็จ                 |
| 400  | Bad Request          | request ผิดรูปแบบ / ข้อมูลไม่ครบ      |
| 404  | Not Found            | ไม่พบ route หรือ resource             |
| 409  | Conflict             | ข้อมูลชนกัน เช่น ซ้ำ                  |
| 422  | Unprocessable Entity | รูปแบบถูก แต่ข้อมูลไม่ผ่าน validation (ไม่ค่อยได้ใช้เพราะใช้ 400)|


## หมวดใหญ่
| หมวด | ความหมาย   |
| ---- | ---------- |
| 1xx  | กำลังทำ    |
| 2xx  | สำเร็จ     |
| 3xx  | เปลี่ยนทาง |
| 4xx  | Client ผิด |
| 5xx  | Server พัง |

ใน API ใช้จริงแค่ 2xx / 4xx / 5xx
