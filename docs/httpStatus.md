# HTTP Status code
HTTP Status Code คือรหัสที่ server ส่งกลับไปเพื่อบอกผลลัพธ์ของ request ว่าสำเร็จ ล้มเหลว หรือผิดพลาดแบบไหน

### สรุปง่าย ๆ  
|200 | ok | ใช้ดูการสำเร็จง่าย ๆ เช่น อ่านไหนข้อมูลไหม
|201 | created | สร้างสำเร็จไหม
|400 | bad_request | ล้มเหลวหรือไม่
|409 | not_found | หาเจอหรือเปล่า
|422 | unproceesable | entity ข้อมูลถูกไหม (ไม่ค่อยได้ใช้เพราะใช้ 400)

## หมวดใหญ่
| หมวด | ความหมาย   |
| ---- | ---------- |
| 1xx  | กำลังทำ    |
| 2xx  | สำเร็จ     |
| 3xx  | เปลี่ยนทาง |
| 4xx  | Client ผิด |
| 5xx  | Server พัง |

ใน API ใช้จริงแค่ 2xx / 4xx / 5xx
