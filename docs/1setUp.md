# Setup NodeJS 
### การติดตั้ง Node.js
1. ติดตั้ง nodeJS จากเว็บไซต์ แล้วเช็ค node -v
2. สร้างโปรเจค vscode และเริ่มใช้คำสั่ง npm ได้
npm i, npm install

3. app.js เป็นศูนย์การจัดการ route และรัน Server ง่าย ๆ ด้วย ``` node app.js```


### Nodejs Core Modules 
การเขียน JavaScript บน Browser มีข้อจำกัดเรื่องความปลอดภัย ทำให้เราไม่สามารถเข้าถึงไฟล์ในเครื่อง หรือจัดการกับระบบ Network ได้โดยตรง แต่ Node.js ถูกสร้างมาเพื่อใช้งานฝั่ง Server

fs (File System) 
- จัดการไฟล์และโฟลเดอร์ 
- อ่าน เขียน ลบ หรือแก้ไขไฟล์ในเครื่อง

http / https 
- สร้าง Server และ Client 
- ทำให้ Node.js รับส่งข้อมูลผ่านโปรโตคอล HTTP ได้

path 
- จัดการเส้นทางไฟล์ (Path) 
- แก้ไขปัญหา Path เพี้ยนเมื่อย้ายโปรเจกต์ไปรันบน OS ต่างกัน

os 
- ดึงข้อมูลระบบปฏิบัติการ 
- เช็กสเปคเครื่อง เช่น RAM, CPU, หรือ Temp directory

events 
- จัดการ Event-Driven
- ช่วยให้เขียนโปรแกรมแบบรอรับเหตุการณ์ (Observer Pattern)

stream 
- จัดการข้อมูลต่อเนื่อง 
- แก้ไขปัญหา RAM เต็มเวลาอ่านไฟล์ขนาดใหญ่มากๆ
